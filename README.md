## 📖 About

DiscoVerse is a modern Discord integration plugin for Minecraft servers built with Paper API and JDA.

It provides a seamless bridge between Minecraft and Discord while offering advanced account linking, chat synchronization, embeds, role synchronization, slash commands, webhooks, and much more.

---

## ✨ Features

- 🔗 Minecraft ↔ Discord Account Linking
- 💬 Two-way Chat Bridge
- 🎨 Beautiful Discord Embeds
- 👥 Discord Role Synchronization
- ⚡ Slash Commands
- 🌐 Webhook Support
- 📡 Bot Presence Management
- 🔒 Secure Verification Codes
- 📁 Modular Configuration System
- 🧩 Built for Paper API
- 🚀 Optimized Performance

---

## 📦 Requirements

- Java 21+
- Paper 1.21+
- Discord Bot Token

---

## 📥 Installation

1. Download the latest `DiscoVerse.jar`
2. Place it inside your server's `plugins` folder.
3. Start the server.
4. Configure the generated configuration files.
5. Restart the server.

---

## ⚙️ Configuration

DiscoVerse automatically generates configuration files on first startup.

Example files:

- config.yml
- discord.yml
- modules.yml
- embeds.yml
- messages.yml
